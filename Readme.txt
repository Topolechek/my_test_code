Первый HTTP-эндпоинт принимает следующую информацию:
x: int
y: int
оператор (+,-,*,/)
На выходе возвращает идентификатор задачи.

Второй HTTP-эндпоинт принимает идентификатор фоновой задачи, полученный в ответе на первый эндпоинт.
На выходе возвращает результат выполнения задачи.

Третий HTTP-эндпоинт не принимает никаких данных на вход.
На выходе возвращает список задач с их статусами.

Для использования 1й ручки, нужно выполнить HTTP POST запрос по адресу "/task" с телом запроса в формате JSON, содержащим параметры "x", "y" и "оператор".
Для использования 2й ручки, нужно выполнить HTTP GET запрос по адресу "/task/{task_id}" где {task_id} это идентификатор задачи, полученный в ответе на первый запрос.
Для использования 3й ручки, нужно выполнить HTTP GET запрос по адресу "/tasks".
