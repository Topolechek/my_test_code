Первый HTTP-эндпоинт принимает следующую информацию:
x: int
y: int
оператор (+,-,*,/)
На выходе возвращает идентификатор задачи.

Второй HTTP-эндпоинт принимает идентификатор фоновой задачи, полученный в ответе на первый эндпоинт.
На выходе возвращает результат выполнения задачи.

Третий HTTP-эндпоинт не принимает никаких данных на вход.
На выходе возвращает список задач с их статусами.
